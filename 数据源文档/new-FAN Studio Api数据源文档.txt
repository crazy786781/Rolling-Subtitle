=========================服务器默认推送消息部分================================

服务器推送消息
初始数据 (initial)

{
  "type": "initial",
  "data": { ... }
}
数据更新 (update)

{
  "type": "update",
  "data": { ... }
}
心跳包 (heartbeat)

{
  "type": "heartbeat",
  "ver": "1.1.0",
  "id": "uuid",
  "timestamp": 1630000000000
}
客户端请求
查询数据 (query)

"query" 或 { "type": "query" }
响应示例：

{
  "type": "query_response",
  "data": { ... }
}
心跳检测 (ping)

"ping" 或 { "type": "ping" }
服务器响应：

{ "type": "pong", "timestamp": 1630000000000 }
注意事项
无效路径会立即关闭连接
客户端需自行实现断线重连逻辑
数据格式与对应业务接口一致
长时间无操作建议主动发送心跳

字段	类型	说明
